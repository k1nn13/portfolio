ICARUS 10

Technical Artist | Programmer | 3D Artist

Overview:

Icarus 10 is an adventure puzzle platformer where the player takes control of a scientific vessel hurtling towards the sun.

Gameplay & Features:

My main roles in the project involved designing and texturing all assets, animation, technical art and level design. Strong communication and consistent version control was key to devlivering the project on time.

  • Original asset design and texturing.
  • UV unwrapping and texture maps optimized for WEBGL platform.
  • Particle systems FX.
  • Custom shaders using shader graph.
  • Custom character controller using rigidbody system.
  • Character animation featuring blend trees.
  • Play Game on Itch.io

Modelling & Texturing:

Using Maya and Substance painter all the assets where created for the project in a short time scale with low poly design a high priority to optimize performance in VR.
