VR 1984:

Technical Artist | Programmer | 3D Artist

Overview:

Based on George Orwells 1984 this solo project explores a day in the life of Winston Smith working for the Ministry of Truth.

Gameplay & Features:

The game is based on escape room style puzzles and allows the user to interact within the virtual enviroment with a narrative applied through a diegetic UI built into the assets guiding the player through the experience.



  • Original asset design and texturing.
  • Lighting design to enhance mood and draw attention to the user.
  • Particle systems designed to create smoke and fog.
  • Custom shaders using shader graph.
  • Game manager to control logic and state of play.
  • Character Animation.
  • Puzzles
  • Custom user interface.
  • VR interaction through sockets.
  • Original sound design and audio.

Modelling & Texturing:

Using Maya and Substance painter all the assets where created for the project in a short time scale with low poly design a high priority to optimize performance in VR.
