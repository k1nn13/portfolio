Character Rigging

Modelling and Animation

From conceptual art and through research I have created a 3D character mesh including a rig which can be exported and animated to an fbx file for a game engine. This project also includes a written report detailing creation and design processes. The demo video displays a fully rendered scene displaying the character movement featuring key properties of animation such as anticipation, follow through and stretch. Learning to effectively use AutoDesk Maya in a short time frame provided a challange which i enjoyed and designing and building models has become a new passion which I look forward to exploring and developing skills to produce high quality 3D assets to specification.
