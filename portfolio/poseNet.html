Pose Net

Machine Learning for Creative Practice

Using ml5.js poseNet I implemented a set of interactive works of generative art developed alongside a dance student from Trinity Laban Conservatoire of Music and Dance. The main feature uses particle systems and takes inspiration from Craig Reynolds Boids developing autonomous agents using the steering behaviour 'seek' to find positions of the body. Implementeing a simple GUI allows the user to control speed and force of attraction creating a variation in effect. From the initial demo I continued to develop this project and created a range of generative pieces with the final demos showcased at Confucius Exhibition Goldsmiths where I interacted with the audience allowing them to use the demo on a large projected screen as well as integrate the visuals during dance performances.
