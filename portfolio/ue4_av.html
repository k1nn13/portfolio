Live Audio-Visual Performance

Using UE4 I developed a custom live audio visual performance tool exploring audio reactive generative design and shaders. During the research phase of this project I experimented with different frameworks and libraries including MAX for Live, Super Collider and openFrameworks. Experimenting with UE4 inspired me and I enjoyed the challange of learning and developing in a new enviroment. As UE4 had not been taught during the course I took it on myself to learn how to use and create a project in the engine. One major skill I have learnt is to transfer understanding of programming knowledge to different languages. The Demo features original audio written and performed using Ableton Live.


This project showcases many of my skills focusing on generative design, understanding event driven logic and implementing communication between devices.

Key features of the project include:

  • Live audio analysis and smoothing in UE4.
  • Recieve and process OSC data from Ableton Live to UE4.
  • Generative Design with audio interaction.
  • Niagara Particle System with audio interaction.
  • Reactive HLSL custom shaders (.usf) exploring ray marching.
