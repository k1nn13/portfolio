Moon Lander:

Unity | Game Dev

Overview:

My venture into game development led to the creation of an engaging game inspired by the classic Lunar Lander.

This project marked my initiation into Unity, where I harnessed the power of the Universal Render Pipeline (URP) shader system to craft visually compelling environments. The game boasts a dynamic turret system, adding strategic depth to the gameplay. With a carefully designed user interface and an intricate state machine governing various game states, this project demonstrates my ability to seamlessly blend creative concepts with technical expertise, providing players with an immersive Lunar Lander experience.

Features:

  • Play Game on Itch.io
  • Universal Render Pipeline.
  • Turret state machine system with ray casting.
  • Generative Design.
