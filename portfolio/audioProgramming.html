Audio Programming

C++ / Open Frameworks

Using openFrameworks I developed a number of demo applications exploring audio programming techniques.


Audio Analysis & Ray Marching

Using audio analysis techniques to smooth frequency values and logic to trigger envelopes at frequency thresholds creates smooth animation when applied to shaders uniforms.

Review Code for Audio Analysis and Ray Marching

Audio Synthesis

Exploring amplitude modulation, frequency modulation and additive synthesis to create complex and rich waveforms. Added features include XML to load and save custom GUI presets, Midi input for keyboard control, and audio visualisation

Review Code for Audio Synthesis

Midi Sequencing

Storing the pressed keyboard notes and creating a custom button class to develop a basic audio sequencer which triggers synthesised sounds.

Review Code for Midi Sequencing

Audio Visual

By applying custom vertex and fragment shaders with audio interaction this audio visual project displays how to create animation from changes within audio.

Review Code for Audio Visual

Point Cloud Mandlebulb

Appliying the envelope which triggers the synthesised sound to also animate the mandlebulb creates a smooth animation and can be used to develop audio visual projects.

Review Code for Point Cloud Mandlebulb
